O que são Clients?

Clients é o lado do cliente ou Client Side. É a parte de quem está consumindo a informação na web, através de computadores, smartphones, tablets ou outros dispositivos por meio da internet. Este lado também é conhecido como Front-end e se conecta e extrai informações de um servidor ou Server que é o conceito que veremos a seguir.

O que são Servers?

Server ou Server Side é o lado do servidor. É ali que ficam armazenados os arquivos, e informações que saõ exibidos ou utilizados no Front-end conforme já vimos anteriormente. Podem ser computadores que guardam informações dentro de redes locais como por exemplo no caso de empresas que tem um "computador central" para guardar arquivos, programas, etc. Normalmente precisam de uma grande capacidade de processamento e armazenamento devido à possível alta quantidade de dispositivos conectados e às informações nele contidas.

Tipos comuns de servidores

  • Servidores de E-mails

    • Os servidores de e-mail são, provavelmente, os servidores mais comuns.

    São eles os responsáveis pelos serviços de hospedagem disponíveis hoje no mercado e neles são armazenados dezenas de milhões de informações em forma de e-mails.

    Sem estes servidores nenhum e-mail pode ser enviado ou recebido na sua caixa de entrada, uma vez que é através deles que a transferência de informações entre um remetente e um destinatário é feita.

  • Servidores em Nuvem

    • O serviço de armazenagem em nuvem tornou-se muito popular nos últimos anos, mas, para que ele seja possível, é necessário um servidor na nuvem.

    Com ele, diferentes tarefas podem ser executadas e armazenadas remotamente, sem a necessidade de que haja um servidor dentro da empresa que o utilizará.

  • Servidores de Bancos de Dados

    • Os servidores de banco de dados são aqueles utilizados para melhorar o desempenho do processamento e transferência de dados de redes que recebem um grande número de requisições.

    Serviços de armazenagem, por exemplo, são grandes utilizadores de servidores de bancos de dados, porque precisam arquivar com segurança grande número de dados e informações.

  • Servidores Web

    • Os servidores web, assim como os servidores de e-mail, são bastante comuns e servem para processar e executar aplicações que disponibilizam conteúdo na internet.

O que é o HTML?

HTML é um acrônimo para HyperText Markup Language (linguagem de marcação de HiperTexto)

Como o próprio nome sugere, é uma linguagem de marcação para páginas web e que podem ser interpretadas pelo navegador.

É composto por tags ou marcações para definir o tipo de texto que está sendo escrito, se é título, parágrafo, citações, listas ou até mesmo imagens, por exemplo. É reconhecido por ser a estrutura de uma página web.

Tipos de listas em HTML

Temos três tipos de listas em HTML; Listas Ordenadas, Listas Não ordenadas e Listas de definição

As listas ordenadas são demarcadas por ordem numérica, em sequência, por exemplo:

  1. item 1
  2. item 2
  3. item 3

Ou também

  1. item 1
  2. item 1
  3. item 1

A Tag utilizada para se escrever listas ordenadas é a <ol>.

Ainda existem outras formas de se escrever listas ordenadas, você conseguirá mais detalhes no site oficial do W3C

As listas não ordenadas são demarcadas por símbolos, por exemplo:

  • item 1
  • item 2
  • item 3

A Tag utilizada para se escrever listas não ordenadas é a <ul>.

Nas listas de definição nós usamos as tags <dt> e <dd> e tem uma demarcação diferente das demais, veremos sobre estas em um outro artigo.
